格式

《mark》 定义带有记号的文本

属性：全局属性

事件：支持常用事件

《meter》 定义度量衡。仅用于已知最大和最小值的度量

属性：form 值：number 描述：规定 meter 元素所属的一个或多个表单

属性：high 值：number 描述：规定被界定为高的值的范围

属性：low 值：number 描述：规定被界定为低的值的范围

属性：max 值：number 描述：规定范围的最大值。

属性：min 值：number 描述：规定范围的最小值。

属性：optimum 值：number 描述：规定度量的最优值。

属性：value 值：number 描述：必需。规定度量的当前值

属性：支持全局属性 值：number 描述：必需。规定度量的当前值

事件：支持基础事件

《progress》 定义运行中的任务进度（进程）

属性：max 值：number 描述：规定需要完成的值

属性：value 值：number 描述：规定进程的当前值。

属性：支持全局属性 值：number 描述：必需。规定度量的当前值

事件：支持基础事件

《time》 定义一个日期/时间

属性：datetime 值：datetime 描述：规定日期/时间。另一种方式，则是由元素的内容给定日期/时间。

属性：支持全局属性 值：number 描述：必需。规定度量的当前值

事件：支持基础事件

表单

《form》 定义一个 HTML 表单，用于用户输入

属性：accept-charset 值：character_set 描述：规定服务器可处理的表单数据字符集

属性：action 值：URL 描述： 规定当提交表单时向何处发送表单数据。

属性：autocomplete 值：on or off 描述：规定是否启用表单的自动完成功能

属性：enctype 值：application/x-www-form-urlencoded or multipart/form-data or text/plain 描述：规定在向服务器发送表单数据之前如何对其进行编码。（适用于 method="post" 的情况）

属性：method 值：get or post 描述：规定用于发送表单数据的 HTTP 方法。

属性：name 值：text 描述：规定表单的名称。

属性：novalidate 值：novalidate 描述：如果使用该属性，则提交表单时不进行验证。

属性：target 值： _blank or _self or _parent or _top 描述：规定在何处打开 action URL。

事件：支持基础事件

《input》 定义一个输入控件

属性：accept 值：audio/* video/* image/* MIME_type 描述：规定通过文件上传来提交的文件的类型。 (只针对type="file")

属性：alt 值：text 描述：定义图像输入的替代文本。 (只针对type="image")

属性：autocomplete 值：on or off 描述：autocomplete 属性规定 input 元素输入字段是否应该启用自动完成功能。

属性：autofocus 值：autofocus 描述：属性规定当页面加载时 input 元素应该自动获得焦点。

属性：checked 值：checked 描述：checked 属性规定在页面加载时应该被预先选定的 input 元素。 (只针对 type="checkbox" 或者 type="radio")

属性：disabled 值：disabled 描述：disabled 属性规定应该禁用的 input 元素

属性：form 值：form_id 描述：form 属性规定 input 元素所属的一个或多个表单

属性：formaction 值：URL 描述：属性规定当表单提交时处理输入控件的文件的 URL。(只针对 type="submit" 和 type="image")

属性：formenctype 值：application/x-www-form-urlencoded or multipart/form-data or text/plain 描述：属性规定当表单数据提交到服务器时如何编码(只适合 type="submit" 和 type="image")

属性：formmethod 值：get or post 描述：定义发送表单数据到 action URL 的 HTTP 方法。 (只适合 type="submit" 和 type="image")

属性：formnovalidate 值：formnovalidate 描述：formnovalidate 属性覆盖 form 元素的 novalidate 属性。

属性：width 值：pixels 描述：width 属性规定 input 元素的宽度。 (只针对type="image")

属性：height 值：pixels 描述：规定 input元素的高度。(只针对type="image")

属性：list 值：datalist_id 描述：属性引用 datalist 元素，其中包含 input 元素的预定义选项。

属性：max 值：number date 描述：属性规定 input 元素的最大值

属性：min 值：number date 描述：属性规定 input元素的最小值。

属性：maxlength 值：number 描述：属性规定 input 元素中允许的最大字符数。

属性：multiple 值：multiple 描述：属性规定允许用户输入到 input 元素的多个值(只适合 type="email" 和 type="file")

属性：name 值：text 描述：name 属性规定 input 元素的名称

属性：pattern 值：regexp 描述：pattern 属性规定用于验证 input 元素的值的正则表达式

属性：placeholder 值：text 描述：placeholder 属性规定可描述输入 input 字段预期值的简短的提示信息

属性：readonly 值：readonly 描述：readonly 属性规定输入字段是只读的

属性：required 值：required 描述：属性规定必需在提交表单之前填写输入字段

属性：size 值：number 描述：size 属性规定以字符数计的 input 元素的可见宽度

属性：src 值：URL 描述：src 属性规定显示为提交按钮的图像的 URL。 (只针对 type="image")

属性：step 值：number 描述：step 属性规定 input 元素的合法数字间隔

属性：value 值：text 描述：指定 input 元素 value 的值

属性：type 值：button checkbox color date datetime datetime-local email file hidden image month number password radio range reset search submit tel text time url week 描述：type 属性规定要显示的 input 元素的类型

属性：支持全局属性

事件：支持基础事件

《textarea》 定义多行的文本输入控件

属性：autofocus 值：autofocus 描述：规定当页面加载时，文本区域自动获得焦点

属性：cols 值：number 描述：规定文本区域内可见的列数

属性：disabled 值：disabled 描述：规定禁用文本区域

属性：form 值：form_id 描述：定义文本区域所属的一个或多个表单

属性：maxlength 值：number 描述：规定文本区域允许的最大字符数

属性：name 值：text 描述：规定文本区域的名称

属性：placeholder 值：text 描述：规定一个简短的提示，描述文本区域期望的输入值

属性：readonly 值：readonly 描述：规定文本区域为只读

属性：required 值：required 描述：规定文本区域是必需的/必填的

属性：rows 值：number 描述：规定文本区域内可见的行数

属性：wrap 值：hard or soft 描述：规定当提交表单时，文本区域中的文本应该怎样换行

属性：支持全局属性

事件：支持基础事件

《button》 定义按钮

属性：autofocus 值：autofocus 描述：规定当页面加载时按钮应当自动地获得焦点

属性：disabled 值：disabled 描述：规定应该禁用该按钮

属性：form 值：form_id 描述：规定按钮属于一个或多个表单

属性：formaction 值：URL 描述：规定当提交表单时向何处发送表单数据。覆盖 form 元素的 action 属性。该属性与 type="submit" 配合使用

属性：formenctype 值：application/x-www-form-urlencoded or multipart/form-data or text/plain 描述：规定在向服务器发送表单数据之前如何对其进行编码。覆盖 form 元素的 enctype 属性。该属性与 type="submit" 配合使用

属性：formmethod 值：get or post 描述：规定用于发送表单数据的 HTTP 方法。覆盖 form 元素的 method 属性。该属性与 type="submit" 配合使用

属性：formnovalidate 值：formnovalidate 描述：如果使用该属性，则提交表单时不进行验证。覆盖 form 元素的 novalidate 属性。该属性与 type="submit" 配合使用

属性：formtarget 值：_blank _self _parent _top 描述：规定在何处打开 action URL。覆盖 form 元素的 target 属性。该属性与 type="submit" 配合使用

属性：name 值：name 描述： 规定按钮的名称

属性：type 值：button reset submit 描述：规定按钮的类型

属性：value 值：text 描述：规定按钮的初始值。可由脚本进行修改

属性：支持全局属性

事件：支持基础事件

《select》 定义选择列表（下拉列表)

属性：name 值：name 描述：定义下拉列表的名称

属性：required 值：required 描述：规定用户在提交表单前必须选择一个下拉列表中的选项

属性：defer 值：defer 描述：规定当页面已完成解析后，执行脚本（仅适用于外部脚本）

属性：src 值：URL 描述：规定外部脚本的 URL

属性：type 值：MIME-type 描述：规定脚本的 MIME 类型

属性：支持全局属性

《embed》 定义了一个容器，用来嵌入外部应用或者互动程序（插件）

属性：height 值：pixels 描述：规定嵌入内容的高度

属性：width 值：pixels 描述：规定嵌入内容的宽度

属性：src 值：URL 描述：规定被嵌入内容的 URL

属性：type 值：MIME_type 描述：规定嵌入内容的 MIME 类型。注：MIME = Multipurpose Internet Mail Extensions。

属性：支持全局属性

事件：支持基础事件

《object》 定义嵌入的对象

属性：data 值：URL 描述：规定对象使用的资源的 URL

属性：form 值：form_id 描述：规定对象所属的一个或多个表单

属性：height 值：pixels 描述：规定对象的高度

属性：width 值：pixels 描述：规定对象的宽度

属性：name 值：name 描述：为对象规定名称

属性：type 值：MIME_type 描述：规定 data 属性中规定的数据的 MIME 类型

属性：usemap 值：#mapname 描述： 规定与对象一同使用的客户端图像映射的名称

属性：支持全局属性

事件：支持基础事件

《param》 定义对象的参数

属性：name 值：name 描述：定义参数的名称（用在脚本中）

属性：value 值：value 描述：描述参数值

属性：支持全局属性

事件：支持基础事件
